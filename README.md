# Apps

A small workspace containing example app scaffolds.

## 📦 Included apps

### CLI (Python)

- Directory: `cli/`
- Run:

```bash
python -m cli.app greet --name Alice
```

### Web (Node + Express)

- Directory: `web/`
- Run:

```bash
cd web
npm install
npm start
```

### Cecil (Robot Control)

- Directory: `cecil/`
- Run:

```bash
cd cecil
npm install
npm start
```

Then open: http://localhost:4000
